# PWA icons

Add these files for the installable PWA manifest:

- **icon-192x192.png** – 192×192 px
- **icon-512x512.png** – 512×512 px

Use a single 512×512 image and scale it down for the 192px version.  
You can generate them with tools like [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) or [RealFaviconGenerator](https://realfavicongenerator.net/).
